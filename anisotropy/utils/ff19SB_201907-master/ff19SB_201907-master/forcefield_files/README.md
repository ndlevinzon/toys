# ff19SB force field files (bundled)

These files come from the **AMBER ff19SB** release (2019-07). The `anisotropy` package loads **`amino19.lib`** by default for partial charges in `parameterize_mesh.py`.

**Required for a minimal GitHub clone:** keep this entire `forcefield_files/` directory.

**Not required at runtime:** the parent tarball’s `test/` tree (excluded via `.gitignore`).

Upstream: University of California San Francisco / AMBER force fields. Redistribute only in compliance with the original ff19SB license and citation requirements.
